📌 Sobre

O menu projetos é uma aplicação desenvolvida para centralizar, organizar e facilitar o acesso a todos os meus projetos em um único ambiente.

A proposta é funcionar como um painel principal (dashboard), permitindo visualizar, acessar e gerenciar rapidamente aplicações desenvolvidas ao longo da minha jornada como desenvolvedor.

Esse projeto reflete não apenas organização, mas também minha evolução técnica e meu compromisso com boas práticas de estruturação e escalabilidade.

🎯 Objetivo

Criar uma solução simples, eficiente e escalável para:

Centralizar todos os projetos em um único local

Melhorar a organização pessoal

Facilitar o acesso rápido às aplicações

Permitir expansão futura com novas funcionalidades

🛠️ Funcionalidades

📁 Listagem dinâmica de projetos

🔗 Acesso direto ao repositório ou aplicação

🏷️ Organização por categorias/tecnologias

📌 Destaque para projetos principais

📊 Estrutura preparada para expansão
